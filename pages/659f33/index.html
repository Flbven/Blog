# 基本使用

使用git init初始化仓库，执行后文件夹中会多出一个.git目录用于管理版本库。

$ git init
1

Git中存在几个名词：

  1. 工作区：在电脑里实际操作的文件夹
  2. 暂存区(stage)：从工作区提交出来，并暂时放在此处
  3. 版本库：由暂存区的内容提交(commit)过来

相关命令：

$ git add [file1] [file2] [dir] # 添加文件/目录到暂存区
$ git commit [file] [-m message] # 提交文件到版本库
$ git status										# 查看状态
